# DISC Test (HTML)

Project ini berisi beberapa versi halaman tes DISC berbasis HTML yang bisa dijalankan langsung di browser.

## Fitur

- Beberapa varian halaman tes DISC (standar dan dengan timer)
- Dapat dijalankan tanpa build tools
- Cocok untuk demo cepat di lokal

## Struktur Project

- `index.html`: halaman utama versi terbaru
- `disc-test.html`: halaman tes DISC versi dasar
- `disc-test-timer.html`: halaman tes DISC dengan timer
- `disc-test-timer-persist.html`: halaman tes DISC dengan timer yang mempertahankan status
- `.gitignore`: mengabaikan file sistem seperti `.DS_Store`

## Cara Menjalankan

1. Buka salah satu file HTML langsung di browser (disarankan `index.html`).
2. Atau jalankan lewat web server lokal (misalnya XAMPP) lalu akses folder ini.
