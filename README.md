# DISC Test (HTML)

Project ini berisi halaman tes DISC berbasis HTML yang bisa dijalankan langsung di browser.

## Fitur

- Tampilan tes DISC dalam satu halaman (`index.html`)
- Dapat dijalankan tanpa build tools
- Cocok untuk demo cepat di lokal

## Struktur Project

- `index.html`: halaman utama tes DISC
- `.gitignore`: mengabaikan file sistem seperti `.DS_Store`

## Cara Menjalankan

1. Buka `index.html` langsung di browser.
2. Atau jalankan lewat web server lokal (misalnya XAMPP) lalu akses folder ini.
